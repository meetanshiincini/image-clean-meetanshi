# image-clean-meetanshi
Magento 2 Image Clean by Meetanshi allows admin to delete the non used product and category images in a single click! It's official now that Google considers page load speed as a ranking factor! Anything that affects the page load speed in the Magento 2 store must be optimized. Meetanshi's Magento 2 Image Clean extension helps admin to optimize the store performance by removing the unused product and category images from the media folder in the Magento 2 store. The Magento 2 store with large database needs to be cleaned from time to time as some of its data may be outdated or useless. The extension helps admin to remove such non-used product or category images from the media folder in the Magento 2 store backend.
